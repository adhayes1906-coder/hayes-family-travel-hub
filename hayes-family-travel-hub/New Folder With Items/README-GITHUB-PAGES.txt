# Hayes Family Travel Hub - GitHub Pages Upload Folder

This folder is ready to upload to a GitHub Pages repository.

Recommended repository name:
hayes-family-travel-hub

Files in this folder:
- index.html
- manifest.json
- sw.js
- icon-192.png
- icon-512.png
- .nojekyll

Upload steps:
1. Open the GitHub repository you want to use.
2. Choose Add file, then Upload files.
3. Upload every file from this folder.
4. Commit the changes.
5. In repository Settings, open Pages.
6. Set Source to Deploy from a branch.
7. Select main and /root, then save.

Notes:
- This is a static app. There is no build step.
- Replace index.html later to update the hub.
- If a phone shows an older version, refresh once or remove and re-add the home-screen app.
- Show times and port details should still be confirmed in the Royal Caribbean app close to sailing.
